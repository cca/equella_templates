This is a wrapper around a few actions that should make updating the CORE Waiver taxonomy a little quicker. The name and structure of the files in the "lib" folder must be maintained for it to work, but otherwise it can be zipped up and moved around. Files in the data folder are just backups; only the most recent CSV needs to be saved.

All you should need to do is double-click "run.command" and follow the prompts. The only manual part is downloading the report from Informer.
